Temario CCNA 2023-2025

El examen Cisco Certified Network Associate (CCNA) prueba tus conocimientos y habilidades relacionados con los fundamentos de redes, acceso a redes, conectividad IP, servicios IP, fundamentos de seguridad y automatización y programabilidad.

Cisco Certified Network Associate

Certificación reconocida mundialmente que valida tus habilidades para administrar y optimizar las redes actuales.

Fundamentos de Redes

  • Modelos de redes (OSI y TCP/IP)
  • Descripción general de los componentes de red
  • Conexiones físicas y topologías
  • Señalización y transmisión
  • Protocolos de comunicación
  • Flujo de datos y encapsulamiento

Ethernet y Conmutación

  • Operación de switches de capa 2
  • Configuración básica de switches
  • VLANs y trunking
  • Spanning Tree Protocol (STP)
  • EtherChannel
  • Seguridad de puerto

Direccionamiento IP

  • IPv4: estructura y notación
  • Cálculo de subredes
  • VLSM y CIDR
  • IPv6: características y formato
  • Configuración de direcciones IP
  • Diagnóstico de problemas de dirección IP

Enrutamiento IP

  • Conceptos básicos de enrutamiento
  • Enrutamiento estático
  • Protocolos de enrutamiento (OSPF)
  • Enrutamiento entre VLANs
  • First Hop Redundancy Protocols
  • Solución de problemas de enrutamiento

Servicios de Red

  • NAT/PAT
  • DHCP
  • Servicios DNS
  • NTP
  • QoS básica
  • Configuración de servicios de red

Seguridad de Red

  • Conceptos de seguridad
  • Listas de control de acceso (ACL)
  • Seguridad de puerto
  • VPN e IPsec básicos
  • Autenticación, Autorización y Contabilidad

Redes Inalámbricas

  • Conceptos básicos de WLAN
  • Estándares 802.11

  • Autenticación y seguridad WLAN

Automatización y Programabilidad

  • Conceptos de automatización de red
  • Comparación de métodos tradicionales vs automatizados
  • Controladores e interfaces (API, REST)
  • Protocolos de configuración (NETCONF, RESTCONF)
  • Formatos de datos (JSON, XML, YAML)
  • Herramientas de automatización (Python)

¿Por qué obtener la certificación CCNA?

La certificación CCNA es una de las más reconocidas en la industria IT y proporciona los conocimientos fundamentales necesarios para una carrera exitosa en redes. Al obtener esta certificación demostrarás tus habilidades en:

  • Implementación y administración de soluciones de red
  • Identificación y resolución de problemas de conectividad
  • Mejora de la seguridad en infraestructuras de red
  • Automatización y programación de redes modernas
  • Comprensión de tecnologías emergentes como SD-WAN, cloud networking y virtualización

Este temario está diseñado para cubrir todos los temas necesarios para aprobar el examen CCNA 200-301, válido hasta 2025.
